# TD_RESTAPI

A RESTful API built with FastAPI and MongoDB.

# Features, stack

- Python, with `FastAPI` and `pymongo` library
- MongoDB, as database
- Docker, for deployment

# API Usage

1. Clone the repository
2. Start the server :

```bash
$ docker-compose up -d
```

3. Go to [localhost:3000/docs](http://localhost:3000/docs)
